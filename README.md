# Automation-Tools

A collection of **custom-built security automation tools** developed to support **penetration testing and defensive security workflows** in **authorized, simulated lab environments**.

This repository serves as a personal **learning and research workspace**, focusing on automating repetitive security tasks while reinforcing core concepts in:
- Reconnaissance
- Asset discovery
- Surface mapping
- Tool chaining
- Security workflow design

All tools are designed with **ethical use, transparency, and controlled environments** in mind.

Each tool in this repository follows a few core principles:

- **Automation over repetition**  
- **Read-only reconnaissance before exploitation**
